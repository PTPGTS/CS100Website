Lessons Learned in CS 100

In CS 100 I was introduced to the basics of computer science and how computers actually solve problems, which helped me understand what is happening behind the scenes when we run programs or use different types of software. I learned how to think more logically by breaking problems into smaller, clear steps using algorithms and pseudocode, and this made it easier to approach assignments in a more organized way. The class also gave me practice with simple programming concepts and showed me how code, data, and hardware all connect to each other. Another important lesson from CS 100 was how to use new tools and programs on my own, including reading instructions carefully, fixing errors, and not getting discouraged when something did not work the first time. These experiences taught me to be more patient and independent as a learner, which I know will help me in future computer science courses and in my career. Overall, CS 100 gave me a solid starting point and helped confirm that I really want to keep going in computer science.
